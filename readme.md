Разработаем Windows Forms приложение на языке Visual C# для чтения файла структурно-параметрической базы (СПБ).

В результате работы программа нам читает СПБ файл.  Следующим этапом разработки элементов программного обеспечения обработки данных структурно-параметрической базы является поиск элементов заданного типа. 
В структурно-параметрической базе может быть задано 14 базовых элемента форма. Произведем поиск этих элементов в файле SPB и выводим их на экран с помощью textbox. 
Разработаем программу, которая будет производит поиск элементов в СПБ базе и выводить существует ли такой элемент в базе или нет.
