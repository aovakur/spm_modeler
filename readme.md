Разработаем Windows Forms приложение на языке Visual C# для чтения файла структурно-параметрической базы (СПБ).
Разработку приложения для чтения структурно-параметрический базы будем вести на языке C# с помощью NET Framework. NET Framework  — программная платформа, выпущенная компанией Microsoft в 2002 году. Основой платформы является общеязыковая среда исполнения Common Language Runtime (CLR), которая подходит для разных языков программирования. Функциональные возможности CLR доступны в любых языках программирования, использующих эту среду.
В структурно-параметрической базе может быть задано 14 базовых элемента форма. Произведем поиск этих элементов в файле SPB и выводим их на экран с помощью textbox.
